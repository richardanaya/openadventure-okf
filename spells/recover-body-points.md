# Recover Body Points

| Field | Value |
| --- | --- |
| **Range**: | Reach |
| **Target**: | One character |
| **DS**: | 2 (base Difficulty Score) + Target’s total number of Body Wounds |
| **Casting Time**: | 6 seconds |
| **Duration**: | Instantaneous |
| **Components**: | Two tea leaves |

On a successful Spellcasting Action Check, the Target recovers a number of Body Points equal to the Effect of the spell, up to the target’s maximum points.

## Design Notes

This is the standard [Recovery Action](/actions/recovery.md) performed using magic instead of mundane tools and skills. This version is to recover Body Points, but just swap out Body in the spell with Mind or Spirit to change the type of points recovered.
