# Open Adventure: Fantasy Core Rules v1.0

An Open Knowledge Format bundle of the Open Adventure: Fantasy Core Rules.

# Contents

* [Open Adventure Fantasy in Open Knowledge Format](README.md) - Explains this repository's conversion of the Open Adventure: Fantasy Core Rules into an Open Knowledge Format bundle.
* [About](about/) - Publication details, licensing guidance, and introduction.
* [Core Rules](rules/) - Core mechanics, character options, equipment, procedures, and guidance.
* [Backstory Generation](backstory/) - Optional d100 tables for character history, identity, values, and motivations.
* [Action Library](actions/) - Individually addressable example game actions.
* [Challenge Library](challenges/) - Individually addressable pre-built challenges.
* [Spell Library](spells/) - Individually addressable pre-built spells.
* [NPC Library](npcs/) - Individually addressable NPC and adversary statblocks.
