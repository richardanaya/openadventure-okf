# Guidance for Adventure Guides

This section contains tips, tricks, and resources to help Adventure Guides run amazing *Open Adventure* games. It is not a step-by-step guide to running adventures, but rather a toolbox filled with advice for both experienced and aspiring Adventure Guides. Topics covered include guidance for running Session Zero, tips for setting Challenge Points and Difficulty Scores, and a deeper dive into adjudicating spells and magic.
