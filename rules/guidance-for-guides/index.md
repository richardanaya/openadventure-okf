# Guidance for Adventure Guides

* [Guidance for Adventure Guides](overview.md) - A toolbox of advice for running Open Adventure games.
* [Session Zero](session-zero.md) - Establish comfort, consent, setting expectations, materials, and character creation guidelines.
* [The Rule of Common Sense](rule-of-common-sense.md) - Use common sense when rules are incomplete or do not fit the situation.
* [Adjudicating Challenges](challenges.md) - Assign, conceal, and improvise Challenge Points.
* [Adjudicating Actions](actions.md) - Resolve player intent by considering what, how, and why before applying rules.
* [Setting Difficulty Scores](difficulty-scores.md) - Set base Difficulty Scores and Drawbacks consistently.
* [Adjudicating Stunts](stunts.md) - Identify and resolve Actions with multiple outcomes or effects.
* [Adjudicating Spellcasting](spellcasting.md) - Set spellcasting Difficulty Scores, modifiers, casting times, and limits.
