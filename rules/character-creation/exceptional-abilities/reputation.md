# Reputation

## Primary Benefit

+1 Benefit to Action Scores for Presence Checks when dealing with individuals who are from the same culture or have the same background.

## Secondary Benefit

+1 Benefit to Action Scores for Presence Checks when dealing with individuals who have the same background.
